# ⌨️ Keyboard Shortcuts Reference

Quick reference for all keyboard shortcuts in the enhanced UI.

---

## Message Input

| Shortcut | Action |
|----------|--------|
| `Enter` | Send current message |
| `Shift + Enter` | Insert new line in message |

---

## Navigation & Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` (or `Cmd + K` on Mac) | Clear chat history |
| `Ctrl + L` (or `Cmd + L` on Mac) | Load/Reload models |
| `Ctrl + E` (or `Cmd + E` on Mac) | Open export conversation dialog |
| `Ctrl + D` (or `Cmd + D` on Mac) | Toggle dark/light mode |

---

## In-App Help

Click the **⌨️ keyboard icon** in the header to view shortcuts anytime!

---

## Tips

1. **Focus the input:** Click in the message box or press `Tab` to navigate
2. **Multi-line messages:** Hold `Shift` while pressing `Enter`
3. **Quick clear:** `Ctrl+K` is fastest way to start fresh
4. **Theme toggle:** `Ctrl+D` works from anywhere in the app

---

## Browser Shortcuts (Still Work!)

These standard browser shortcuts also work:

| Shortcut | Action |
|----------|--------|
| `F12` | Open Developer Tools |
| `Ctrl + R` | Reload page |
| `Ctrl + +` | Zoom in |
| `Ctrl + -` | Zoom out |
| `Ctrl + 0` | Reset zoom |
| `F11` | Toggle fullscreen |

---

## Mouse Shortcuts

Don't forget these quick actions:

- **Hover over messages** to see the copy button
- **Click model name** in header to see current selection
- **Click status dot** to see connection info

---

**Pro Tip:** Learn 2-3 shortcuts you'll use most (like `Ctrl+K` and `Ctrl+D`) for a much faster experience!
